Features

Everything you need to manage audio devices efficiently.

⚡

Global Hotkeys

Assign keyboard shortcuts to any audio device and switch instantly from anywhere — no window focus needed.

🎧

All Devices

Enumerate and manage all input and output audio devices on your system in one place.

🎨

7 Theme Presets

Choose from Catppuccin Mocha, Light, Nord, Dracula, Solarized Dark, Tokyo Night, and Gruvbox.

🔌

System Tray

Runs quietly in the background. Minimize to tray and access settings from the context menu.

💾

Persistent Config

Your device bindings and theme preference are saved automatically as JSON — no setup on restart.

💻

Cross-Platform

Native support for Windows (MMDevice API) and Linux (PulseAudio + X11). Built with Qt 6.

Installation

Get started in seconds on your platform.

Windows Installer

Download the installer from the latest release and run it.

1. Download QuickSnapAudio-Setup.exe from Releases
2. Run the installer
3. Launch from Start Menu or Desktop

Debian / Ubuntu

Download the .deb package and install:

sudo dpkg -i quicksnapaudio_*.deb

Build from Source

Requires CMake 3.16+, Qt 6.x, and a C++17 compiler.

cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --parallel
